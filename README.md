# Jchannel
高専版2chプロジェクト

## これは何なの？
インターネット老人会なら誰でもが知っている「2ch」を高専生がそれっぽく作って運用してみようという計画です。

## 名前の由来
超単純です。情報系の高専生なので、頭文字の「J」と、「2ch」をリスペクトし、「ch」の部分だけもらいました。
くっつけたら「Jch」です。
